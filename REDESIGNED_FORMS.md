## 🌈 Color Palette

```css
/* Primary */
Orange: #F97316 (orange-500)
Orange Hover: #EA580C (orange-600)

/* Backgrounds */
Main BG: linear-gradient(to bottom right, #111827, #111827, #1F2937)
Card BG: rgba(31, 41, 55, 0.5) with backdrop-blur
Input BG: rgba(17, 24, 39, 0.5)

/* Text */
Primary: #FFFFFF (white)
Secondary: #9CA3AF (gray-400)
Muted: #6B7280 (gray-500)

/* Borders */
Default: #4B5563 (gray-600)
Focus: #F97316 (orange-500)

/* Shadows */
Button: rgba(249, 115, 22, 0.3)
Button Hover: rgba(249, 115, 22, 0.5)
Card: 2xl shadow
```

## 📐 Spacing & Sizing

- **Card Padding**: 2rem (8)
- **Input Height**: 3.5rem (py-3.5)
- **Button Height**: 4rem (py-4)
- **Form Gap**: 1.25rem (space-y-5)
- **Border Radius**: 
  - Cards: 1.5rem (rounded-3xl)
  - Inputs/Buttons: 0.75rem (rounded-xl)
